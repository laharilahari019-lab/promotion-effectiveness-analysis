# promotion-effectiveness-analysis
📊 Promotion Effectiveness Trend Analysis
🎯 Objective

To analyze whether promotions generate real demand or artificially inflate discounts.

📁 Dataset

Contains:

Date
Product ID
Original Price
Discounted Price
Promotion Flag
⚙️ Steps Performed
Calculated daily discount
Identified non-promotional days
Calculated baseline discount
Compared with promotional discounts
Created line chart
📈 Key Insights
Promotions increase discount levels
Some promotions are artificially inflated
Baseline pricing remains stable
Helps identify genuine vs fake promotions
🛠 Tools Used
Excel
